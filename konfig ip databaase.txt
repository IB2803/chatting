1. mysql -u root -p
atau 
1. mysql -u root

Misalnya kamu ingin mengizinkan semua IP di jaringan lokal (192.168.%):(tinggal ganti ip yang dimau)
2. GRANT ALL PRIVILEGES ON *.* TO 'root'@'192.168.%' IDENTIFIED BY '' WITH GRANT OPTION;

Agar MySQL menerapkan perubahan:
3. FLUSH PRIVILEGES;

Untuk memastikan root sudah punya akses:
4. SELECT Host, User FROM mysql.user WHERE User = 'root';
